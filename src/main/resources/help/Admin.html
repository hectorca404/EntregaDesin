Menú Administrador

Desde este menú, el administrador puede gestionar diferentes aspectos del sistema.

  • Crear Parada: Permite registrar nuevas paradas en el sistema, asignando un responsable y sus credenciales.
  • Cerrar Sesión: Finaliza la sesión actual del administrador y regresa a la pantalla de inicio de sesión.

Para realizar una acción, simplemente haz clic en el botón correspondiente.
